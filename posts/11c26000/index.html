浏览器插件分享——Wetab

发表于2022-10-28|更新于2022-10-28
|字数总计:1.6k|阅读时长:4分钟|阅读量:|  点赞:0

前言

浏览器作为上网冲浪使用频率最高的工具之一，如何使浏览器变得更加高效也是许多网友热衷于去折腾的一个话题。

但是最近浏览器更新之后，启动时无法打开 infinity 插件作为启动页面，单击主页按钮倒是没问题。我尝试卸载重装的过程中发现了开发团队的另一款插件——Wetab。

插件当主页有什么用

现在的浏览器基本都支持自定义主页拨号盘，你可以很便捷地通过拨号盘直达想去的网站，甚至 Edge 浏览器同样可以有一个很 nice 的主页。

对比下来，我们可以看到第三方插件主页不一定比浏览器自带的主页更美观（但是vivaldi的默认主页好丑），但是插件的个性化程度比浏览器原生的好用了不知多少倍。Edge 浏览器的原生主页还有信息流内容展示，这也并不符合我对“纯净”的需求。

如果你只是想能快捷进入想去的网站，浏览器原生主页和书签栏完全可以满足你的需求；如果你希望自定义化更高一点，第三方插件或许值得一试；如果你还需要更多功能，那么你真的可以试试 Wetab。

Wetab能干嘛

相较于 infinity，wetab在拨号盘图标之外新增了“小组件”功能。实不相瞒，就目前来说，wetab 支持的小组件不算非常丰富，但是却有几个好用的小组件，加上它可以更随心所欲地排列图标和组件，看起来比 infinity 更加美观大方——如果要用比喻来形容的话，wetab 事实上有点像 windows 8的开始菜单，像又不太像。

其中，热门网站几乎每个都有多个图标可以选择，冷门网站通常会自动获取网站的favicon。而小组件都可以调节大、中、小三种不同的展示样式。

不过这些都不是这个插件最亮眼的点，接下来看看其中几个好用的小组件。

天气

此前使用的 infinity 插件也可显示天气，Edge 浏览器原生的主页在左上角也有显示。但是如果你想查看更多天气或者今日详情，需要额外点击才行。而 Wetab 的小组件可以通过卡片的方式直接展示出来。卡片设置为“小”的时候仅展示今日天气和温度，为“中”时会额外展示今日的湿度和风速等内容，为“大”时还会额外展示其它几天的天气。点进卡片会看到更多具体的内容。

热搜

热搜小组件可以在你打开浏览器之后向你展示今日的热点新闻，可选择的源有微博热搜、百度热搜、知乎热榜等等共计十个，你可以选择你更喜欢的一个源作为卡片默认展示内容。

待办清单

前面提到过，浏览器是上网冲浪的高频使用软件，因此，一打开浏览器就可以看到待办清单也一定程度上能提醒自己还有什么事情要做。

待办清单的添加和完成都很简单，完成待办事项后只需要点击一下事项，即可自动标记为完成。如果需要清理已完成时间，也只需要点一下就够了。此外，待办清单还支持分组功能，便于区分不同紧急程度和不同方面的事件。

笔记

Vivaldi 浏览器自带笔记功能，并且也支持云端同步。但是由于它根本没有样式可言，所以我通常只是用来将那些需要付费复制文字的网页复制到笔记里，绕过关注公众号或者付费这一环节。

Wetab 的笔记功能支持 Markdown 语法。常见的字体样式，代码块，引用块，表格，行内公式等等全都支持。笔记小组件默认的笔记文档里一一展示了它支持的 Markdown 特性。

除了这些，我还发现一个特别实用的功能——上传图片。笔记小组件会自动将上传或粘贴的图片上传到云端，在任何地方都可以直接引用外链。

经测试，2.9Mb大小的图片加载用时只需要1秒，不同的网络环境可能有所区别，我在单位的移动网环境下测试只要500多毫秒。虽然速度不错，使用也很便捷，但是不建议作为图床滥用，这样可能会导致开发组的成本增加，以致于最后被限制使用。

其它

其它的小组件还包括倒计时、生日和纪念日等等，尤其是生日和纪念日功能，在有对象之后你会发现它是真的实用——没有对象怎么办？家人的生日你记得住吗⚡

在官网展示的截图中，还有豆瓣电影小组件，在目前使用的版本中没有找到这一个功能，期待未来能开发出更多实用的小组件吧。

不想用插件？

如果你觉得浏览器不必要为了主页额外安装一个插件，你同样可以通过网页版来使用 Wetab。在功能和界面上，网页版和插件版没什么区别，插件版相对来说可能就是将正在浏览的网址添加到主页更加便捷一些吧。

由于插件和网页都是通过账户同步，并且在做了修改之后（无论是笔记还是待办清单的编辑器内容更改）都会自动同步。所以如果你也正在使用 infinity 或者同类插件，不妨尝试一下 Wetab，如果你从未使用过插件作为浏览器主页，你也可以花几分钟尝试一下，说不定就爱上了呢。

在线版体验地址：https://wetab.link
